### Trading Approach

**What I Trade:**
- **Penny Stocks / Small Caps** - Main focus. Momentum setups, structure reads, high relative volume plays
- **Futures** - ICT methodology, automating fair value gaps and liquidity concepts in Python
- **MetaTrader Bots** - Multiple automated strategies with strict risk controls

**How I Trade:**
- **Penny Stock Framework** - Gap scanners, float rotation, premarket momentum, overhead resistance levels
- **ICT Concepts** - Fair Value Gaps, liquidity grabs, order flow for futures
- **Quantitative Methods** - Making discretionary patterns systematic through backtesting and statistics

**The Goal:** Apply 18+ years of building robust systems to trading. Make it repeatable, scalable, data-driven.
